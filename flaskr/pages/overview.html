There are three type of input features

  • Objective: factual information
  • Examination: results of medical examination;
  • Subjective: information given by the patient.

Age

Objective feature   int(days)

Height

Objective feature      int(cm)

Weight

Objective feature   float(Kg)

Gender

Objective feature categorical code
